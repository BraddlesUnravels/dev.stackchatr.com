# Skills Showcase

Foundation for a skills portfolio site built with Qwik City, TypeScript, Bun,
Tailwind CSS v4, ESLint, and Prettier.

## Requirements

- Bun

## Getting Started

```shell
bun install
bun dev
```

## Scripts

- `bun dev` - SSR dev server
- `bun start` - dev server and open browser
- `bun build` - production build
- `bun preview` - preview production build
- `bun build.types` - typecheck
- `bun lint` - ESLint
- `bun fmt` - Prettier write
- `bun fmt.check` - Prettier check

## Styling

Tailwind v4 is wired through the Vite plugin and `src/global.css`.

## Qwik Integrations

Use `bun qwik add` to add adapters like Netlify or Cloudflare.
